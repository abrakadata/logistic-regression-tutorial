# Logistic Regression Interactive Tutorial

An interactive, from-scratch tutorial that teaches Logistic Regression using:
- a single-page HTML walkthrough with visual demos,
- live probability and decision boundary visualizations,
- a training-loss animation,
- and a companion Python implementation.

## Project Structure

- `Logistic Regression Interactive Tutorial/Logistic Regression.html` - interactive tutorial page
- `Logistic Regression Interactive Tutorial/Logistic Regression From Scratch.py` - plain Python implementation from scratch

## Quick Start

1. Open `Logistic Regression Interactive Tutorial/Logistic Regression.html` in your browser.
2. Navigate sections from the top menu.
3. Use the sliders and training demo to explore model behavior.

## What You Will Learn

- Why binary classification needs logistic regression instead of linear regression
- Sigmoid function intuition and behavior
- Model equation and prediction thresholding
- Binary cross-entropy loss
- Gradient descent updates for weights and bias
